## 📖 简介

河南经贸智能助手是基于 [Cherry Studio](https://github.com/CherryHQ/cherry-studio) 开源项目定制开发的 AI 智能助手平台，专为河南经济贸易技师学院师生打造。

## ✨ 主要功能

### 🎓 教育专用助手

| 分类 | 助手 | 功能 |
|------|------|------|
| **教师助手** | 📚 备课助手 | 教案设计、课件大纲、教学活动 |
| | 📝 出题助手 | 试题生成、组卷、考核方案 |
| | ✅ 批改助手 | 作业批改、评语生成 |
| | 🔬 教研助手 | 论文写作、课题申报 |
| **学生助手** | 🎓 学习辅导 | 知识讲解、答疑解惑 |
| | 📖 作业助手 | 作业指导、写作帮助 |
| | 🎯 职业规划 | 就业指导、简历优化 |
| | 🔧 技能助手 | 实训指导、操作规范 |
| **行政助手** | 📄 公文写作 | 通知、报告、总结 |
| | 🗓️ 会议助手 | 议程安排、纪要整理 |
| | 📊 数据统计 | 数据整理、报表制作 |

### 🛠️ 核心功能

- 💬 **AI 对话** - 支持多种大语言模型
- 📚 **知识库** - 文档问答、资料检索
- 🌐 **翻译功能** - 中英文互译
- 📁 **文件处理** - 支持 PDF、Word、图片等
- 📝 **笔记功能** - 记录和整理信息

## 📥 下载安装

前往 [Releases](https://github.com/houcong/hnjm-inteligence/releases) 页面下载对应平台的安装包：

| 平台 | 下载 |
|------|------|
| Windows | `河南经贸智能助手-x.x.x-x64-setup.exe` |
| macOS | `河南经贸智能助手-x.x.x-arm64.dmg` |
| Linux | `河南经贸智能助手-x.x.x-x64.AppImage` |

## 🚀 快速开始

1. 下载并安装应用
2. 打开应用，进入「设置」→「模型服务商」
3. 配置 AI 模型（推荐使用 DeepSeek 或通义千问）
4. 开始使用 AI 助手

## 💡 推荐模型配置

| 模型 | 提供商 | 特点 |
|------|--------|------|
| DeepSeek-V3 | DeepSeek | 性价比高、中文优秀 |
| 通义千问 | 阿里云 | 稳定可靠 |
| 文心一言 | 百度 | 中文理解强 |

## 📋 使用须知

- 本平台仅供河南经济贸易技师学院内部教学使用
- 请勿将 AI 生成内容直接用于正式文件，需人工审核
- 遵守学院相关规定，合理使用 AI 工具
