## 二、功能定制建议

### 1. 教育场景专用助手

建议预置以下 AI 助手：

#### 教师助手
- **备课助手**: 生成教案、课件大纲、教学设计
- **出题助手**: 根据知识点生成试题、组卷
- **批改助手**: 作业批改建议、评语生成
- **教研助手**: 论文润色、课题申报

#### 学生助手
- **学习辅导**: 知识点讲解、习题解答
- **作业助手**: 写作指导、格式规范
- **职业规划**: 就业指导、简历优化
- **技能助手**: 实训操作指导

#### 行政助手
- **公文写作**: 通知、报告、总结
- **会议助手**: 会议纪要、议程安排
- **数据分析**: 报表解读、趋势分析

#### 实训助手
- **操作指导**: 设备操作步骤
- **安全规范**: 安全注意事项
- **故障排查**: 常见问题解决

### 2. 知识库集成建议

```
知识库分类                内容
─────────────────────────────────────
学院制度库               规章制度、管理办法、工作流程
课程资料库               各专业课程资料、教材、课件
实训手册库               实训操作手册、设备说明书
试题资源库               历年试题、模拟题、题库
就业指导库               就业政策、面试技巧、行业资讯
```

### 3. 功能模块取舍建议

#### 建议保留
- ✅ AI 对话（核心功能）
- ✅ 知识库（文档问答）
- ✅ AI 助手（预置角色）
- ✅ 翻译功能
- ✅ 文件处理

#### 建议移除或隐藏
- ❌ 绘图功能（非教育核心需求）
- ❌ Mini 程序（简化界面）
- ❌ 划词助手（减少复杂度）
- ❌ 部分高级设置

## 五、部署方案

### 方案 A：桌面端分发（推荐初期使用）

```
优点：部署简单、离线可用
缺点：需要逐台安装、更新麻烦

实施步骤：
1. 打包 Windows/Mac 安装包
2. 内置统一的 API 配置
3. 通过内网分发安装
4. 适合：机房电脑、教师办公电脑
```

### 方案 B：内网服务器部署（推荐长期使用）

```
优点：数据不出校园、统一管理、无 API 费用
缺点：需要服务器资源、运维成本

架构：
┌─────────────┐     ┌─────────────┐
│  客户端      │────▶│  内网服务器  │
│  (桌面应用)  │     │  (Ollama)   │
└─────────────┘     └─────────────┘

服务器配置建议：
- CPU: 16核以上
- 内存: 64GB以上
- GPU: NVIDIA RTX 4090 或更高（可选）
- 存储: 500GB SSD
```

### 方案 C：混合部署

```
内网模型：处理敏感数据、日常问答
云端 API：处理复杂任务、高级功能

优点：兼顾安全性和能力
```

## 七、开发环境准备

### 1. 环境要求

```bash
Node.js >= 22.0.0
Yarn 4.x
Git
```

### 2. 本地开发

```bash
# 克隆项目
git clone https://github.com/houcong/hnjm-inteligence.git
cd hnjm-inteligence

# 安装依赖
yarn install

# 启动开发模式
yarn dev

# 构建检查
yarn build:check

# 打包应用
yarn build:win    # Windows
yarn build:mac    # macOS
yarn build:linux  # Linux
```

### 3. 目录结构说明

```
src/
├── main/              # Electron 主进程
│   ├── services/      # 后端服务
│   └── ipc.ts         # 进程通信
├── renderer/          # React 前端
│   └── src/
│       ├── components/  # UI 组件
│       ├── pages/       # 页面
│       ├── store/       # 状态管理
│       ├── config/      # 配置文件 ⭐ 助手配置
│       └── i18n/        # 国际化 ⭐ 界面文字
└── preload/           # 预加载脚本
```
